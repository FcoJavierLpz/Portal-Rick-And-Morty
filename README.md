# Portal Rick And Morty

### Requirements

- Node.js 18.18.2
- yarn
- Git
- ViteJs
- Typescript
- React 18
- zustand
- Chakra Ui

### Setup

For dependencies use:

```bash
$ yarn install
```

For install git hooks into git repository

```bash
yarn prepare
```

⚠️ Git Commits: Follow the conventional commits defined in the file .conmmitlintrc

### Development

For develop cycle use:

```bash
$ yarn dev
```

### Testing static

**Linters & Formatters**

Find lint issues

```bash
$ yarn lint
```

Find format issues

```bash
$ yarn format
```

### Unit Testing

```bash
$ yarn test
```

### Deploy

```bash
$ yarn build && yarn preview
```
